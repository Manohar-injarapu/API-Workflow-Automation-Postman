# API-Workflow-Automation-Postman

->project demonstrates end-to-end API automation using Postman by chaining multiple requests:
Login (Generate Token), Create User, Update User, Delete User

->The automation uses environment variables to dynamically pass data (token, user ID) between requests.

Step 1: Create a New Environment
Step 2: Create a New Collection
Step 3: Request 1 — Login and Extract Token
Step 4: Request 2 — Create User and Extract ID
Step 5: Request 3 — Update User
Step 6: Request 4 — Delete User
Step 7: Run the Automated Flow
